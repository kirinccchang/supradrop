What gets checked
error SUPRA_NONEXISTENT — supra note N where footnote N doesn't exist in the document.
error SUPRA_FORWARD — supra note N where N ≥ current footnote number (forward reference).
error SUPRA_TO_SHORT_CITE — supra note N where footnote N itself starts with a short citation (Id. or supra), not a full citation.
warn SUPRA_NO_PINCITE — supra note N without a following pincite; Bluebook requires "supra note N, at [page]".
error ID_NO_PRECEDING — Id. with no preceding citation to reference.
warn ID_AMBIGUOUS — Id. following a footnote that appears to cite multiple sources; the referent is ambiguous.
warn SEE_GEN_NO_PAREN — See generally without an explanatory parenthetical; Bluebook strongly prefers one (e.g., (arguing that...)).
warn QUOTE_NO_PINCITE — Direct quote without an apparent pincite; Bluebook R. 3.2 requires a pincite for all quoted material.
